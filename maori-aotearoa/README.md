# Māori (Aotearoa New Zealand) Prompts

This directory contains AI prompts specifically designed for the Aotearoa New Zealand context, focusing on Te Tiriti o Waitangi, Tikanga Māori, and relevant legislation.

## Prompts Available
- **[General Review Prompt](general-review-prompt.md)**: Use this for general documents, products, or policies.
- **[Workplace Review Prompt](workplace-review-prompt.md)**: Use this when you have an internal organisational Māori strategy or Te Tiriti framework to align with.

## Key Frameworks Referenced
- **Te Tiriti o Waitangi**: The foundational partnership between the Crown and Māori.
- **Tikanga Māori**: Māori customary values and practices (e.g., Manaakitanga, Kaitiakitanga).
- **Waitangi Tribunal Reports**: Specific sectoral reports (e.g., WAI 2575 for Health).
- **Te Mana Raraunga**: Māori Data Sovereignty principles.

## Regional Context
These prompts are designed to be "Mana Whenua aware," meaning they prompt you to identify the specific Iwi or Hapū of the region where your work takes place.
