# Contributing to Indigenous Cultural Alignment Prompts

Thank you for being here. This repository is intended to be a safe, respectful, and educational space for learning and contribution.

I welcome feedback, corrections, and new prompt contributions, including prompts for additional countries and Indigenous groups.

## Contribution Principles

- Respect Indigenous self-determination, local authority, and community protocol.
- Keep content practical, culturally safe, and educational.
- Do not write final content on behalf of users; prompts should guide thinking and review.
- Use reputable and verifiable sources (community authorities, government frameworks, trusted research institutions).

## Before You Open a PR

1. Open an issue first with your proposed change.
2. Wait for maintainer confirmation before submitting large updates.
3. Keep proposals clear: what you want to change, why it matters, and which sources you used.

## Testing Expectations

If you are adding or changing prompts, please test your changes before submission.

1. Clone the repository.
2. Use the `tests/` folder to understand the evaluation approach and expected quality bar.
3. Run available tests locally where relevant.
4. In your pull request, include a short note on what you tested and what you observed.

## Pull Request Steps

1. Reference the approved issue.
2. Fork the repository and create a branch.
3. Make focused changes (prompt content, sources, or tests).
4. Submit a pull request with:
	- What changed
	- Why it changed
	- What testing you completed

## Maintainer Review

All submissions are reviewed by the maintainer before merge. Accepted changes go through an additional internal quality and cultural safety review process aligned with this project's methodology.

For project background, see `README.md` and `METHODOLOGY.md`.
